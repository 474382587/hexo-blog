Binary Search

Binary Search 的前提条件是 1. 有序 2. 数组
满足以后，即可取中间值判断与目标值是否相等， 如不相等，再取中间值对比，此时中间值为 prev中间值和最低值、最高值 中间的值

mid = (mid + high)/2
mid = (low + mid)/2

Time: O(logN)
